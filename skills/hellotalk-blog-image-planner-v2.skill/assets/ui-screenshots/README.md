# UI Screenshots Library

Place your raw HelloTalk app screenshots in this folder. These will be referenced when the AI generates `ui-screenshot-modified` type images.

## Recommended Files to Add

Capture these screens from your HelloTalk app and save them with the exact filenames below:

```
matching-screen.png         — Partner discovery / matching interface
chat-interface.png          — 1-on-1 text chat with bubbles
voice-message-ui.png        — Voice message recording / playback UI
voicerooms-list.png         — List of available voice rooms
voicerooms-active.png       — Inside an active voice room
moments-feed.png            — Moments (social feed)
profile-with-tags.png       — User profile showing interest/profession tags
correction-popup.png        — Native speaker correction interface
ai-partner-chat.png         — AI conversation partner UI
translation-inline.png      — In-chat translation feature
```

## How to Capture

1. Use a test account (not your real personal account)
2. Take clean screenshots without any real user data visible
3. Save as PNG, mobile resolution (1080 × 1920 or similar)
4. Place in this folder

## What the AI Will Do

When this skill triggers, the AI will:
1. Identify which UI screenshot is needed for the current article
2. Generate a modification prompt referencing the relevant file from this folder
3. Output the prompt for you to feed into Codex / GPT-4o along with the source screenshot

## Privacy Reminder

Even if these are your own screenshots, ensure they don't capture other real users' data (names, photos, messages). Use a test account for clean captures.
