> **截流人群**：寻求 international community 的成年学习者
> **文章语言**：英文（关键词面向英文搜索，正文用英文写）

**关键词清单（共 5 个，主题统一）**

1. meet foreign friends online

## 一、写作总要求

1. **语气**：像一个"有经验的过来人在分享"，不是广告腔。第一人称实测、可信度优先。
2. **人称**：尽量用 we / us / our（营造同为学习者的共鸣），少用命令式的 you。
3. **字数**：Pillar 2500-3500 词；App 清单 2000-2500 词；How-to 2000-2800 词。
4. **结构**：H2 / H3 分层清晰；短段落（2-4 句）；重点（痛点、功能名、结论）加粗。
5. **必备模块**（头部文章普遍都有，我们也必须做到）：
   - 开篇用真实场景 / 痛点切入，不要开篇推产品；
   - App 类必须有**置顶对比表**；
   - 结尾 **FAQ**（4-6 题，吃长尾、抢 People Also Ask）；
   - **自然内链**（见第三节，重点）。
6. **情感价值是本主题的灵魂**：通篇强调"交到真朋友 / 真实文化连接"，而不仅是"练语言的工具"。

## 三、文章互链要求（集群内链，重点！）

> SEO 文章集群的核心就是**内链**。单篇写得再好，不互链就是一盘散沙，权重无法集中、爬虫也理不清主次。本批 4 篇**必须**按下面的"中心辐射（Hub-and-Spoke）"模型互链。

**模型：**
- **Pillar 是枢纽**：正文里按场景自然链向 A / B / C 每一篇。
- **每篇支撑文回链 Pillar**：开头或结尾用描述性锚文本指回 Pillar。
- **支撑文之间按相关性横链**（见下表）。

**本批互链关系表（用上面规划好的 slug，动笔即预埋，一起上线后即生效）：**

| 从 → 到 | 在什么地方链 | 锚文本示例 |
|---|---|---|
| Pillar → A | 讲"用哪些 App 交友"时 | best apps for meeting international friends |
| Pillar → B | 讲"和母语者对话练习"时 | apps to talk to native speakers |
| Pillar → C | 讲"想说得更像母语者"时 | how to talk like a native English speaker |
| A → Pillar | 开头/结尾回链 | our complete guide to meeting international friends online |
| A → B | 提到语言交换类 App 时 | apps built for talking to native speakers |
| B → Pillar | 开头/结尾回链 | meeting international friends online |
| B → C | 讲"练到更自然"时 | how to talk like a native English speaker |
| B → A | 讲"纯交友向选择"时 | best apps for meeting international friends |
| C → Pillar | 开头/结尾回链 | meet international friends online |
| C → B | 讲"找真人母语者练"时 | apps to talk to native speakers |

**硬性要求：**
- 每篇**至少 3-5 条站内链**，其中**必须有 1 条回链 Pillar**。
- 锚文本用**描述性短语**，禁用 "click here"；不要每条都用一模一样的主词（避免过度优化被判堆砌）。
- **预埋链接**：4 篇用规划好的 slug 互相链接，哪怕暂时还没上线（一起发布后链接即生效）。
- **盘活旧文**：同时链向相关的已上线文（如 `/best-language-exchange-apps`、`/chat-with-foreign-friends`），避免新文成为孤儿页，也给旧文导权重。
- 每篇正文自然处**软链 HelloTalk 官网** `https://www.hellotalk.com/en` 1-2 次（不堆砌）。

## 四、HelloTalk 品牌要求（红线，写手必须遵守）

### 必须自然嵌入的官方数据（择需选用，不要堆砌）
- **70M+ registered users**（全球 7000 万+ 注册用户）
- **200+ countries** / **260+ languages**
- **90% of core features free**（90% 核心功能免费）
- **over 1 billion messages daily**
- 荣誉（保留具体年份名称）：**2017 Google Play Best Social App**；**2024 global Google Play homepage feature**

### 四大功能模块必须有实质描述（可不单独成节，但都要提到）
- **Chat-based learning**：聊天内置翻译、音标 / 转写、朗读、实时语法纠错，不用切换 App。
- **Moments**：公开社区动态，发内容获多位母语者纠错，也可刷母语者真实日常。
- **Voicerooms & Livestreams**：24 小时多人语音房（可先当听众进入）+ 有互动的直播小课。
- **AI learning tools**：AI 发音评分（指出具体问题）、AI 语法纠错（实时带解释）、图片翻译。

### 核心关键词
真实 / 沉浸 / 互动（real / immersive / interactive，自然融入）。

### 竞品规则
- **绝对禁止出现 Tandem**（任何形式都不行）。
- 可对比：**Duolingo、Speaky、Busuu**。对比时突出 HelloTalk 的真人母语者互动、工具集成、免费比例优势。

### 其他红线
- 不得暗示 HelloTalk 是**约会 / 婚恋平台**（本主题尤其小心：定位"交友 / 文化交流 / 语言交换"，不是 dating）。
- 不提骗子 / 诈骗等负面词（安全建议正面表述即可，如"保护隐私、循序渐进、公开场合见面"）。
- 不涉政治、不涉敏感地域、不写"在国内免翻墙 / 无需 VPN 可用"。
- 举例人名用国际化名字（Alex / Sarah / Emma / David），禁用 Xiaoming / Li Hua 等本土占位名。

### 禁 AI 痕迹（全文零容忍）
- **禁止符号**：em dash（—）、箭头（→）、正文分隔线（---）。
- **禁用词**：ultimate / revolutionary / best-in-class / unparalleled / game-changing / cutting-edge / groundbreaking / redefine / seamlessly / effortlessly / robust / utilize / landscape / transforming / transformative / immense / paramount / delve / underscore / elevate。
- **禁止句式**：演讲式（"I can't stress this enough"）、断言式（"The clear winner is…"）、模板标签（"Key takeaway:"）、假设式（"Imagine…"）、Meta 表达（"What's worth noting is…"）。
